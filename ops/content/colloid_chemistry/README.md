# Curated-модуль по коллоидной химии

Пакет дополняет production-ассистента по коллоидной химии тремя темами, для которых исходный курс содержит материал, но текущие канонические карточки недостаточны: модель БЭТ, электрофоретическая подвижность и ζ-потенциал, а также качественная интерпретация энергетического профиля ДЛФО.

Версия пакета: `COLLOID-CURATED-2026-07-R1`. Ревизия R1 уточняет контракт самодостаточности для ζ-задач после production regression.

В curated-слой включены только проверяемые определения, расчётные соотношения, единицы и границы применимости. Пакет не содержит лабораторных процедур, рецептур, аварийных указаний или ответов закрытого тестового банка.

## Состав пакета

- `grounding.md` — student-facing разделы COL-00…COL-03;
- `import-manifest.json` — SHA-256 источников, страницы PDF, manual-verification gates, четыре предложения карточек и пять блюпринтов с рубриками.
- `prompts/` — точный repo snapshot трёх вручную выверенных active system prompts и их production provenance.

## Порядок импорта

1. Сверить SHA-256 `Lektsia_full.pdf` и визуально открыть страницы, перечисленные в манифесте.
2. До импорта привязать ассистента к фактическому курсу Picrete; `course_scope` в этом пакете намеренно пуст, пока course binding не утверждён.
3. Загрузить `grounding.md` как `methodical`, `authority=course_lecture`, `visibility=student`.
4. Создать карточки COL-00…COL-03 по `reference_sheet_proposals` и сохранить source provenance.
5. Заменить `reference_sheet_slugs` блюпринтов на реальные `reference_sheet_ids`; остальные поля `payload` совместимы с `POST /api/assistants/{assistant_id}/templates`.
6. Рубрики использовать при ревью задачи и эталона; сумма каждой равна 10.
7. Создать и активировать версии промптов по инструкции `prompts/README.md`; для всех трёх ролей использовать `target_family=deepseek`.
8. Перед student publish преподаватель подтверждает формулы, обозначения, систему единиц и область применимости модели.

## Жёсткие ограничения

- `Тест.pdf` не является источником этого пакета. Он содержит визуальные пометки ответов, дублирующиеся метки вариантов и непроверенный ключ; до ручной нормализации должен оставаться `unverified + quarantine`;
- формулы нельзя импортировать из text layer `Lektsia_full.pdf`: греческие символы и порядок дробей повреждаются при извлечении;
- диапазон \(p/p_s\) для БЭТ задаётся условием конкретной задачи; нельзя превращать один учебный диапазон в универсальное правило;
- уравнение Гельмгольца–Смолуховского используется только при явно указанном приближении тонкого двойного слоя;
- ζ-задача задаёт либо \(v\) и \(E\), либо готовую \(\mu_e\); в обоих случаях обязательны численные \(\eta\), \(\varepsilon_r\) и \(\varepsilon_0\) в SI;
- эталон не подставляет универсальные константы из памяти и не классифицирует устойчивость по одному \(\zeta\) без курируемого критерия, области применимости и источника;
- ДЛФО-задачи в этом пакете интерпретируют уже заданный энергетический профиль или таблицу, а не проектируют реальную коагуляцию.

При расхождении с действующим курсом приоритет имеет утверждённая преподавателем версия курса.
