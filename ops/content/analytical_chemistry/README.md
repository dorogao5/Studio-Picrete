# Curated-модуль по аналитической химии

Этот пакет дополняет production-ассистента по аналитической химии короткими проверяемыми карточками для тем, которые были заявлены в профиле, но недостаточно представлены в каноническом grounding: качественный и гравиметрический анализ, кулонометрия по закону Фарадея, базовая кондуктометрия и интерпретация данных СЗМ.

Пакет содержит только теорию, расчётные соотношения и контроль доказательности. Он не является лабораторной методикой и не содержит инструкций по приготовлению растворов, проведению реакций, первой помощи или действиям при аварии.

## Состав пакета

- `grounding.md` — student-facing curated-слой ANA-00…ANA-05;
- `import-manifest.json` — источники с SHA-256 и страницами PDF, manual-verification gates, предложения карточек и шесть блюпринтов с рубриками.
- `prompts/` — точный repo snapshot трёх вручную выверенных active system prompts и их production provenance.

## Порядок импорта

1. Сверить SHA-256 источников и повторно открыть перечисленные страницы PDF.
2. Загрузить `grounding.md` как `methodical`, `authority=course_lecture`, `visibility=student`, `course_scope=c31f4f7a-5982-4c06-9f97-ee1f3fb58db6`.
3. Создать карточки из разделов ANA-00…ANA-05 по `reference_sheet_proposals`.
4. Сохранить `source_provenance` из манифеста в описании карточки или в поддерживаемом поле provenance; не оставлять карточку без ссылки на файл и страницы.
5. При импорте блюпринтов заменить `reference_sheet_slugs` на реальные `reference_sheet_ids`; объект `payload` соответствует `POST /api/assistants/{assistant_id}/templates`.
6. Рубрики из `recommended_rubric` использовать при ревью задачи и формировании эталона. Сумма каждой рубрики равна 10.
7. Создать и активировать версии промптов по инструкции `prompts/README.md`; для всех трёх ролей использовать `target_family=deepseek`.
8. До публикации преподаватель отдельно подтверждает формулы, обозначения, единицы и область применимости.

## Жёсткие ограничения

- [аналитика экз.pdf](</Users/doroga/Documents/аналитическая химия/Новая папка с объектами/аналитика экз.pdf>) не является источником пакета и должна оставаться `unverified + quarantine`: в ней есть повреждённые формулы, ненормативный текст и небезопасные фрагменты;
- пособие ИТМО используется только для абстрактных определений дробного/систематического анализа. Страницы с первой помощью, пожарными советами, конкретными реактивами, классификациями на основе токсичных реагентов и лабораторными процедурами не импортируются;
- примеры и числовые данные из источников не становятся каноническими автоматически;
- формулы нельзя копировать из извлечённого text layer: индексы, заряды, греческие символы и единицы сверяются по изображению страницы;
- СЗМ-карточка не заменяет руководство конкретного прибора и не разрешает настройку режима измерения.

Если действующий материал курса расходится с этим пакетом, приоритет имеет утверждённая преподавателем версия курса.
